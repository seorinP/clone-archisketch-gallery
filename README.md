# (주)아키드로우 사의 archisketch의 렌더 이미지 갤러리 페이지 클론코딩


## 사용 기술 및 라이브러리

React, Typescript, Material UI


### `npm start`로 프로젝트 시작 가능

create-react-app으로 만든 프로젝트입니다. npm start 시 [http://localhost:3000](http://localhost:3000) 로 실행됩니다.
